## ReactPizza ­ЪЇЋ

**ReactPizza** рЃљрЃарЃўрЃА рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃърЃарЃЮрЃћрЃЦрЃбрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃЋрЃЦрЃЏрЃћрЃюрЃў **React**РђЊрЃўрЃА рЃерЃћрЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃљрЃЊ.  
рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃгрЃљрЃарЃЏрЃЮрЃљрЃЊрЃњрЃћрЃюрЃА рЃърЃўрЃфрЃўрЃА рЃерЃћрЃЎрЃЋрЃћрЃЌрЃўрЃА рЃАрЃљрЃўрЃбрЃўрЃА рЃАрЃўрЃЏрЃБрЃџрЃљрЃфрЃўрЃљрЃА, JWT рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃЌ рЃЊрЃљрЃфрЃБрЃџрЃў REST APIРђЊрЃЌ.

­ЪїЇ **рЃњрЃљрЃерЃЋрЃћрЃЉрЃБрЃџрЃў рЃЋрЃћрЃарЃАрЃўрЃљ:** [ReactPizza (Render)](https://reactpizza-je52.onrender.com/)

## Рюе рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

### Client side (React)

- рЃЎрЃљрЃбрЃћрЃњрЃЮрЃарЃўрЃћрЃЉрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ рЃърЃўрЃфрЃўрЃА рЃЊрЃљрЃЌрЃЋрЃљрЃџрЃўрЃћрЃарЃћрЃЉрЃљ
- рЃФрЃўрЃћрЃЉрЃљ рЃЊрЃљ рЃАрЃЮрЃарЃбрЃўрЃарЃћрЃЉрЃљ
- рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃљрЃЊ рЃЊрЃљрЃДрЃЮрЃцрЃўрЃџрЃў рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў (Pagination)
- рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃў рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџрЃў рЃърЃўрЃфрЃўрЃАрЃЌрЃЋрЃўрЃА
- рЃЎрЃљрЃџрЃљрЃЌрЃўрЃА рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

### Backend (Flask REST API)

- JWT (RS256 private/public key pair) рЃЊрЃљрЃфрЃБрЃџрЃў рЃљрЃЊрЃЏрЃўрЃюрЃўрЃАрЃбрЃарЃљрЃбрЃЮрЃарЃўрЃА рЃарЃЮрЃБрЃбрЃћрЃЉрЃў
- рЃърЃўрЃфрЃўрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃљ, рЃарЃћрЃЊрЃљрЃЦрЃбрЃўрЃарЃћрЃЉрЃљ рЃЊрЃљ рЃгрЃљрЃерЃџрЃљ
- PostgreSQL рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃљ
- Cloudinary рЃўрЃюрЃбрЃћрЃњрЃарЃљрЃфрЃўрЃљ рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА
- Swagger UI рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљ

### 3. PostgreSQL рЃЉрЃљрЃќрЃљ

### 4. JWT рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃћрЃЉрЃў

JWT-рЃАрЃЌрЃЋрЃўрЃА рЃАрЃљрЃГрЃўрЃарЃЮрЃљ RSA рЃњрЃљрЃАрЃљрЃдрЃћрЃЉрЃћрЃЉрЃўрЃА рЃгрЃДрЃЋрЃўрЃџрЃў:

```
bash
  openssl genrsa -out jwt_private.pem 2048
  openssl rsa -in jwt_private.pem -pubout -out jwt_public.pem
```

рЃцрЃљрЃўрЃџрЃћрЃЉрЃў рЃњрЃљрЃюрЃљрЃЌрЃљрЃЋрЃАрЃћ `backend` рЃЊрЃўрЃарЃћрЃЦрЃбрЃЮрЃарЃўрЃљрЃерЃў.

`.env` рЃцрЃљрЃўрЃџрЃерЃў рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћ:

```
bash
  JWT_PRIVATE_KEY_PATH=./jwt_private.pem
  JWT_PUBLIC_KEY_PATH=./jwt_public.pem
```

## ­ЪћЉ рЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃљрЃфрЃўрЃљ

`backend/.env` рЃцрЃљрЃўрЃџрЃў рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃЊрЃћрЃА:

```
.env
  # Flask
  FLASK_ENV=development
  FLASK_DEBUG=1
  SECRET_KEY=super_secret_key
  SECRET_PASSWORD=super_secret_password

  # Database
  DATABASE_URL=postgresql://user:password@localhost:5432/reactpizza_db

  # JWT
  JWT_PRIVATE_KEY_PATH=./jwt_private.pem
  JWT_PUBLIC_KEY_PATH=./jwt_public.pem
  JWT_ACCESS_TOKEN_EXPIRES_MINUTES=30
  JWT_REFRESH_TOKEN_EXPIRES_MINUTES=1440

  # Cloudinary
  CLOUDINARY_CLOUD_NAME=your_cloud_name
  CLOUDINARY_API_KEY=your_api_key
  CLOUDINARY_API_SECRET=your_api_secret
```
